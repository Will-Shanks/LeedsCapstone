oneCol module¶

Find company names in 1 col pages

oneCol.get_titles(lines)¶

Prints out company names found in given .day files

Parameters

lines (Generator[str, None, None]) – generator that produces the manual line by line

Returns

list of company names found

Return type

list[str]

Note

If a name miss is detected “MISSED COMPANY NAME” is added to the returned name list
