dayToDF module¶

Code to generates a datafram from .day ocr files With example code that iterates through lines of the scaned page

dayToDF.get_df(fn)¶

Reads given day file, and return a dataframe describing it

Parameters

fn (str) – FilePath to a .day file

Returns

Dataframe describing said .day file

with rows: {x,y}{min,max}, col, line, word where col, line, and word give the ordering on the page e.x: col=1,line=2,word=0 is the first word of the third line of

the second column

Return type

pandas.DataFrame

dayToDF.iter_df(df)¶

Generator that iterates row by row over df

Parameters

df (pandas.DataFrame) – Containing columns text, word, line, col

Yields

str – Each str is a line in the df
