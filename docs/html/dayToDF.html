dayToDF module¶

Code to generates a datafram from .day ocr files With example code that iterates through lines of the scaned page

class dayToDF.DayReader(year, **kwargs)¶

Bases: object

Manages complicated day file parsing

Can parse through a years day files, splitting by change in # of cols, not page end. This is potentially usefull for parsing, as sometimes a company info wraps to the next page but it does not appear they will wrap to a different column style page

Note

Will raise an FileNotFoundError if no day files are found
